## How to Use

1. Open `index.html` in your browser to view the CV
2. Customize the HTML with your own information
3. Modify the styling in the `<style>` section to match your preferences
4. Update the favicon by replacing `logo.jpg` with your own image

## Features

- ✨ Clean and minimal design
- 📱 Mobile responsive layout
- 🎨 Professional styling with accent colors
- 🔗 Social media links
- 📊 Organized sections (Skills, Education, Experience, Contact)
- 🌐 SEO-friendly meta tags

## Customization

To customize this CV:

1. Edit the page title and meta descriptions in the `<head>` section
2. Update personal information (name, location, contact details)
3. Add or remove skills
4. Update education and experience sections
5. Add your own social media profile links
6. Modify colors and styling as needed
